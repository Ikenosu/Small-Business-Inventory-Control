This project is a web-based application developed using HTML, CSS, and JavaScript.

It does not require compilation or a standalone executable (.exe) file.

To run the system:
1. Extract the source code folder
2. Open index.html using any modern web browser (Chrome / Edge)
3. The system will run directly in the browser

No executable module is applicable for this project. Rather, the link to start using the application can be found here: https://ikenosu.github.io/Small-Business-Inventory-Control/
